###Prepare:
1. docker
2. download the code source
3. python3.5(only need for testing)
###Install:
    ./go.sh install
###Test:
    ./go.sh test
    ./go.sh sa
###Remote Install
    ./go.sh remote-install [user] [host]
note: You have to provide the Environment variable:
1. POSTGRES_DATABASE
2. POSTGRES_USER
3. POSTGRES_PASSWORD
4. POSTGRES_URL
